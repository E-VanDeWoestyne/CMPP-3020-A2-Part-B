# 🎓 SAIT Enrollment Management System

## 📌 Overview
This project is a prototype **Enrollment Management System** developed as part of a group assignment. The system is designed to help administrative staff manage student enrollment records efficiently.

The application allows users to:
- Add new student records  
- Modify existing student information  
- Remove student records  
- View all enrolled students  

This project demonstrates key programming language concepts such as **variables, data types, binding, referencing, control structures, and subprograms**.

## 🧩 Features
Each student record includes:
- First Name & Last Name  
- Date of Birth  
- Gender  
- GPA (from previous institution)  
- Current Semester  
- Program  
- Number of Courses  

The system supports:
- Dynamic data storage during runtime  
- Record management using appropriate data structures (arrays, lists, etc.)  
- User interaction via console input/output

## ⚙️ Key Concepts Demonstrated

### 🔹 Variables & Data Types
- Use of primitive and non-primitive data types  
- Structured storage of student data  

### 🔹 Data Binding
- Static typing in Part 2 language  
- Dynamic typing in Python  

### 🔹 Control Structures
- Conditional statements (`if`, `switch`)  
- Loops (`for`, `while`)  

### 🔹 Subprograms (Functions/Methods)
- Add student  
- Modify student  
- Delete student  
- Display records  

### 🔹 Referencing & Memory
- Pass-by-value and pass-by-reference  
- Use of arrays/lists and object references
